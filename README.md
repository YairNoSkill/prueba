#  Gestor de Tareas - Prueba Técnica

Este repositorio contiene la solución a la prueba técnica para el puesto de Desarrollador Junior. 
Está construido con HTML, CSS, JavaScript Vanilla y aplicando una arquitectura modular.

##  Demostración
Puedes ver el video del funcionamiento del sistema y la validación de roles aquí:
 [Ver video de demostración en Google Drive](https://drive.google.com/file/d/12ho36W0vi_ZefUbU_h4Z5JzTE1DEbZzi/view?usp=sharing)

##  Entregables incluidos
* Código fuente completo con diseño responsivo (Mobile First).
* Script de Base de Datos (SQL Server) con creación de tablas e inserts de prueba.
* Sistema de autenticación simulada con persistencia en LocalStorage.
* Lógica estricta de roles (Administrador / Usuario Normal).
